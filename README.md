# Gestión de Películas MVC - Proyecto 2do Parcial 
Desarrollo de una Aplicación Web con PHP y MySQL utilizando MVC Y CRUD

## Tecnologías Utilizadas
* Backend: PHP 8+, MySQL
* Frontend: HTML5, CSS3 (Custom Properties, Flexbox), JavaScript Vanilla.
* Arquitectura: MVC (Modelo - Vista - Controlador)

## Funcionalidades
* CRUD completo manejando dos entidades relacionadas (Películas y Directores).
* Validaciones de datos tanto en el lado del cliente (JS/HTML5) como del servidor (PHP).
* Diseño interactivo adaptado de mi proyecto de primer parcial.
